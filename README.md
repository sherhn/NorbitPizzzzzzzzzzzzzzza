# Краткая инструкция по запуску скриптов из папки `build`:

# 

# \## rebuild.ps1

# ```powershell

# .\\build\\rebuild.ps1

# ```

# \- Останавливает контейнеры

# \- Пересобирает образы без кеша

# \- Запускает контейнеры в фоне

# 

# \## rebuild\_image.ps1

# ```powershell

# \# По умолчанию (суффикс "web")

# .\\build\\rebuild\_image.ps1

# 

# \# С указанием суффикса

# .\\build\\rebuild\_image.ps1 -s "db"

# ```

# \- Останавливает контейнеры с удалением volumes

# \- Удаляет указанный образ

# \- Пересобирает все образы без кеша

# \- Запускает контейнеры
