**cart.js** - работа с корзиной:
- Добавление/удаление товаров, управление количеством
- Оформление заказа с адресом доставки
- Синхронизация с сервером (загрузка/сохранение состояния корзины)

**cities.js** - переключение городов:
- Выбор города доставки
- Обновление города в форме заказа

**favorites.js** - избранные товары:
- Добавление/удаление из избранного
- Отображение списка избранных товаров

**languages.js** - переключение языков (интерфейс)

**menu.js** - работа с меню:
- Отображение товаров по категориям (пицца, закуски, напитки)
- Фильтрация по ингредиентам
- Переключение вкладок меню

## Взаимодействие:

**С сервером:** Все скрипты общаются с API по адресу `http://localhost/api/`:
- `cart.js` - endpoints: `/orders/cart`, `/orders/make_order`
- `favorites.js` - endpoints: `/main/favorite`, `/main/get_favorites`  
- `menu.js` - endpoint: `/main/get_products`

**Между скриптами:**
- `menu.js` и `favorites.js` используют общие функции для обновления кнопок избранного
- `cart.js` и `cities.js` синхронизируются по городу доставки
- `menu.js` вызывает функции из `cart.js` при добавлении в корзину

**Пользовательский поток:** Выбор города → Просмотр меню → Добавление в корзину/избранное → Оформление заказа.
